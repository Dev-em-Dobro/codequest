# Fase 1 - Estrutura inicial Next.js

## Objetivo da fase
Subir a base Next.js sem quebrar a aplicacao atual React + Express.

## O que foi entregue
- Scaffold do Next.js criado em next-app/ com App Router.
- TypeScript, Tailwind e ESLint configurados pelo template oficial.
- Provider global de React Query adicionado em next-app/src/app/providers.tsx.
- Layout e pagina inicial ajustados para shell de migracao do CodeQuest.

## Como executar
- cd next-app
- npm run dev

## Resultado esperado
- Aplicacao Next.js sobe localmente.
- Base pronta para iniciar migracao de rotas na Fase 2.
